# Canonical Ownership

This repository is the canonical home for playable BOB games.

## Owns

- `snake/` - canonical Python/tkinter BOB Snake for Windows and browser-adjacent demos.
- `sims/` - canonical BOB SIMS life simulation.
- `parallive/` - canonical threaded life simulation.
- `minecraft/` and `terraria/` - canonical browser minigames.
- `chess/` - canonical browser BOB Chess game: human white vs THE MESH with WORM-sealed moves.
- `kittyverse.py` and `enzoi.py` - canonical tkinter hub/avatar runtime for cross-game identity.

## Does Not Own

- `bob-engine` owns the DOOM/Vortex raycaster engine and DOS COM assembly proof.
- `bob-orchestrator` owns BOB compliance, UiPath, NATS, Bedrock, and Trust Deed orchestration. Any game files in `bob-orchestrator` are legacy hackathon artifacts unless explicitly moved here.

When a playable game needs to change, update this repo first and treat duplicate copies elsewhere as stale until proven otherwise.
